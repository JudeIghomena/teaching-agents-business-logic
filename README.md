# Teaching: Business Case Logic with AI Coding Agents

A living reference library for building business-grade logic and agent workflows
using Claude Code, Codex, and the Anthropic Agent SDK.

Maintained by Jude Ighomena / Janna AI Research Labs.

---

## What This Repo Is

This repo is the authoritative teaching resource for implementing real business
case logic with modern AI coding agents. Every document here is practical first:
concrete code, concrete decisions, concrete trade-offs.

Topics progress from foundational frameworks through advanced agent orchestration.

---

## Structure

```
frameworks/          Core frameworks and mental models for agent-assisted development
requirements/        What agents need to perform at high capability (env, context, tooling)
agents/              Agent design patterns: single agent, multi-agent, HITL, tool-use
business-logic/      Business case examples: validation, routing, pricing, approvals
security/            Security patterns for production agent deployments
examples/            End-to-end worked examples (annotated code + explanation)
```

---

## How to Use

Read in order if you are new to agent-assisted development:

1. `frameworks/` — understand the mental model first
2. `requirements/` — set up your environment correctly
3. `agents/` — learn agent patterns
4. `business-logic/` — apply patterns to real business cases
5. `examples/` — study complete worked examples

---

## Principles

- Every concept is grounded in working code
- Every design decision is explained, not assumed
- Security and correctness are not optional layers — they are built in from the start
- Documents are updated as agent capabilities evolve
